## Version

By running `hanami version` we can see the current version of the framework that we are using.

```shell
$ bundle exec hanami version
v1.3.0
```

There are also `--version` and `-v` aliases.

```shell
$ bundle exec hanami --version
v1.3.0

$ bundle exec hanami -v
v1.3.0
```
